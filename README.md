# Phone-Catalogue-App

## Installation

You can run this repo with this following commands:

```sh
git clone https://github.com/JuanMrtt/Phone-Catalogue-App
```

### Open two terminals:

#### First One:
```sh
cd Phone-Catalogue-App/
cd server
npm i
npm run dev
```

#### Second One:
```sh
cd Phone-Catalogue-App/
cd client
npm i
npm start
```
